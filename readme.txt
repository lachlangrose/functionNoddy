This is the batch (and function) versions of Noddy that simply takes a history file and calculates a block model and gravity and magnetic data from this file.

Usage:

noddy input_history  output_prefix

open source blah blah blah...

Use with care, I haven't tested this particular version of the grav/mag calcs to see if they are correct, but they seem to be...
